## Problem Statement ##

Design a ML model for data provided in 12)ToyotaCorolla.csv which can predict the price of Toyota Corolla based on the other feature values provided. 

## Solution ##
1) Using Python's ML and Plotting libraries, first the data is loaded in Pandas data frame.
2) Exploratory data analysis is performed
3) Data is divided in training and testing sets
4) Data is fit in Linear Regression model which is best suited for the given data. 
5) Complete Python code is provided as jupyter notebook "LinearRegression_ToyotaCorollaPrice.ipynb"
